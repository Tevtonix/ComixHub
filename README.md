# ComixHub

> Тактическая платформа для публикации и чтения комиксов.  
> Авторы загружают главы — читатели оставляют отзывы и оценки.

---

## 1. Описание проекта

**ComixHub** — веб-сервис для публикации и чтения комиксов, построенный на FastAPI. Платформа разделяет пользователей на два типа: **авторы** (публикуют комиксы и главы) и **читатели** (читают, оценивают, комментируют, добавляют в закладки). Интерфейс выполнен в стилистике тактических HUD-экранов 2000-х — монохромная зелёная палитра, моноширинные шрифты, CRT-эффекты.


---

## 2. Основные возможности

### Для всех пользователей
- Просмотр каталога комиксов с постраничной навигацией (12 на страницу)
- Поиск по названию, описанию и имени автора
- Просмотр профилей авторов и читателей
- Чтение глав с постраничным отображением и навигацией между главами

### Для зарегистрированных пользователей
- Регистрация и аутентификация через JWT-токены (cookie)
- Добавление комиксов в закладки и управление ими
- Написание комментариев с оценкой (1–5 ★) к главам
- Удаление собственных комментариев
- Редактирование email в профиле

### Для авторов
- Публикация комиксов с обложкой и описанием
- Добавление, редактирование и удаление глав
- Загрузка страниц глав в форматах: `jpg`, `jpeg`, `png`, `webp`, `gif`
- Удаление комментариев читателей под своими главами
- Панель своих комиксов на странице профиля

## 3. Стек технологий

| Слой | Технология |
|---|---|
| Backend | [FastAPI](https://fastapi.tiangolo.com/) |
| ORM / Модели | [SQLModel](https://sqlmodel.tiangolo.com/) |
| База данных | SQLite |
| Аутентификация | JWT (`python-jose`) + bcrypt |
| Шаблонизатор | Jinja2 |
| Сервер | Uvicorn |
| Frontend | HTML + CSS (vanilla, без фреймворков) |
| Шрифты | Google Fonts: Orbitron, Share Tech Mono, VT323 |

## 5. Переменные окружения

Создайте файл `.env` в корне проекта:

```env
SECRET_KEY=your-super-secret-key-change-in-production
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=10080
```

| Переменная | Описание | Значение по умолчанию |
|---|---|---|
| `SECRET_KEY` | Секретный ключ для подписи JWT-токенов. Обязательно смените в продакшне | `your-super-secret-key-change-in-production-2026` |
| `ALGORITHM` | Алгоритм подписи JWT | `HS256` |
| `ACCESS_TOKEN_EXPIRE_MINUTES` | Время жизни токена в минутах | `10080` (7 дней) |

## 6. Как запустить проект

### Требования

- Python 3.10+
- pip

### Установка и запуск

**1. Клонировать репозиторий**

```bash
git clone https://github.com/your-username/comixhub.git
cd comixhub
```

**2. Создать и активировать виртуальное окружение**

```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# Linux / macOS
python -m venv .venv
source .venv/bin/activate
```

**3. Установить зависимости**

```bash
pip install fastapi uvicorn sqlmodel python-jose bcrypt jinja2 python-multipart pydantic-settings
```

**4. Создать файл `.env`**

```bash
# Windows
copy .env.example .env

# Linux / macOS
cp .env.example .env
```

Откройте `.env` и задайте свой `SECRET_KEY`.

**5. Запустить сервер**

```bash
python run.py
```

Приложение будет доступно по адресу: **http://127.0.0.1:8000**

> База данных `comixhub.db` и папка `static/uploads/` создаются автоматически при первом запуске.

### Остановка сервера

```
Ctrl + C
```
